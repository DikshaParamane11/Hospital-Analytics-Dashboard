# 🏥 Hospital Analytics Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing hospital data to generate meaningful insights using Power BI. The dashboard helps in monitoring patient trends, doctor performance, treatment effectiveness, and financial metrics.

## 🎯 Objectives
- Analyze patient and treatment data
- Track doctor performance
- Monitor revenue and financial growth
- Identify trends and improve decision-making

## 📊 Key Features
- Patient Overview Dashboard
- Doctor Performance Analysis
- Treatment Analysis
- Financial Insights Dashboard

## 📈 KPIs Created
- Total Patients
- Total Doctors
- Total Revenue
- Average Recovery Rate

## 🛠 Tools & Technologies
- Power BI
- Power Query
- DAX

## 🔍 Insights Generated
- Identified top-performing doctors
- Detected low-performing treatments
- Analyzed revenue trends
- Monitored patient recovery patterns

## 📁 Dataset
- 5,000+ hospital records

## 🚀 How to Use
1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Interact with dashboards using filters

## 📌 Conclusion
This project demonstrates how data visualization can help hospitals improve performance and make data-driven decisions.
